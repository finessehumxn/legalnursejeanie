# Jeanie Vatelia — Legal Nurse Consulting

Professional website for Jeanie Vatelia, BSN RN — Legal Nurse Consultant based in California.

## Features

- Full single-page site with sticky nav, mobile hamburger menu
- Hero with stats panel and starting rate ($150/hr)
- Three-track business model (Independent Contract, Subcontract LNC, Blended Partner)
- Transparent pricing / rates section
- Six core services
- Personal story section
- Attorney inquiry form (mailto — no backend required)
- Nurse team application form (mailto — no backend required)
- Fully responsive (mobile, tablet, desktop)
- Blue, purple, nude & black color palette
- Google Fonts: Cormorant Garamond (display) + Inter (body)

## Tech stack

Pure HTML/CSS/JS — zero dependencies at runtime. Uses:
- Google Fonts (Cormorant Garamond + Inter)
- Tabler Icons (CDN)
- `serve` for Railway static hosting

## Deploy to Railway

1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub repo
3. Select this repo — Railway auto-detects `railway.json` and deploys
4. Set a custom domain in Railway settings (optional)

## Deploy to GitHub Pages

1. Push to GitHub
2. Go to repo Settings → Pages → Source: Deploy from branch → `main` / `root`
3. Site will be live at `https://yourusername.github.io/jeanie-vatelia-lnc`

## Local development

```bash
npm install
npm run dev
# → http://localhost:3000
```
